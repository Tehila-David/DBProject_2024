## תאור הארגון
אגף שומה וגבייה הוא הגוף האחראי בעיריית ירושלים על גביית הארנונה והיטלים עירוניים אחרים מהתושבים והעסקים בעיר. תפקידו המרכזי הוא לקבוע את שיעור הארנונה השנתי על בסיס שטח הנכס, ייעודו וערכו, ולגבות את התשלומים מבעלי הנכסים השונים.
